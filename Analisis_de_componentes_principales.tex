\begin{frame}{Análisis de Componentes Principales (ACP)}

\begin{block}{Objetivo}
\begin{itemize}
  \item Reducir la dimensionalidad de la data reteniendo tanta de la variabilidad original como sea posible.
  \vspace{3pt}
  \item Generar un nuevo conjunto de variables no correlacionadas (componentes principales) que retengan la mayor cantidad de variabilidad de la data original.
\end{itemize}
\end{block}

\begin{block}{Observaciones}
\begin{itemize}
  \item Útil cuando las variables están correlacionadas.
  \item Técnica de carácter exploratorio/descriptivo, mas NO probabilístico!
  \item La distribución probabilística de $\Xv=[X_1,X_2,\ldots,X_p]^T$ no es de interés.
  \item Input para otro método (e.g., análisis de regresión).
\end{itemize}
\end{block}

\end{frame}



\begin{frame}{Método}

\begin{block}{Formulación}
\begin{align*}
Y_1 &= a_{11}X_1 + a_{12}X_2 + \ldots + a_{1p}X_p = \av^T_1\Xv\\
Y_2 &= a_{21}X_1 + a_{22}X_2 + \ldots + a_{2p}X_p = \av^T_2\Xv\\
\vdots \\
Y_p &= a_{p1}X_1 + a_{p2}X_2 + \ldots + a_{pp}X_p = \av^T_p\Xv
\end{align*}
\end{block}

\begin{block}{Coeficientes (cargas/pesos)}
\centering
$a_{jk}$: contribución/aporte de la variable $X_k$ en la componente $Y_j$.
\end{block}

\begin{itemize}
  \item Las cargas son escogidas de tal manera que las componentes tengan la \textbf{mayor variabilidad posible} (en orden decreciente) y además sean \textbf{mutuamente incorrelacionadas}.
  \item Para que la variabilidad de las componentes no sea arbitrariamente grande se impone la \textbf{restricción} $\|\av_j\| = 1$.
\end{itemize}

\end{frame}



\begin{frame}{Computo}

\begin{block}{}
Para \textbf{maximizar} una función de varias variables con \textbf{restricciones} se utiliza el método de los \textbf{multiplicadores de Lagrange}:
\begin{itemize}
  \item $\var{Y_j} = \lambda_j$ i.e., $j$-ésimo \textbf{valor propio} de $\S$,
  \item $\av_j = \ev_j$, i.e., $j$-ésimo \textbf{vector propio} de $\S$,
\end{itemize}
donde $\lambda_1 \geq \lambda_2 \geq \ldots \geq \lambda_p$\quad y\quad $\ev_j \cdot \ev_k = 0$.
\end{block}

\begin{block}{}
\centering
$\S\,\ev_j = \lambda_j\,\ev_j$
\end{block}

\begin{block}{Observaciones}
\begin{itemize}
  \item La media de los datos no influye en el proceso (centrar la data).
  \item $\lambda_1 + \lambda_2 + \ldots + \lambda_p = s^2_1 + s^2_2 + \ldots + s^2_p$.
  \item La interpretación de las componentes $Y_j$ no es directa.
  \item El ACP no es invariante a la escala de medición.
  \item Las variables $X_j$ con mayores varianzas tienden a dominar.
  \item La estructura de las componentes depende de la escala de medición.
  \item Usar $\R$ cuando hay problemas de conmensurabilidad.
\end{itemize}
\end{block}

\end{frame}


\begin{frame}{En la práctica}

\begin{block}{Cuántas componentes son necesarias?}
\begin{itemize}
  \item Graficar $\lambda_j$ frente a $j$ (\textit{scree diagram}) y excluir aquellas componentes que no provoquen un cambio drástico en la curva.
  \item Escoger las primeras componentes que acumulen entre 70\% y 90\% de la variabilidad.
  \item Excluir la componente $Y_j$ si $\lambda_j<\bar{\lambda}$.
  \item Usando $\R$, excluir la componente $Y_j$ si $\lambda_j < 0.7$.
\end{itemize}
\end{block}

\begin{block}{Puntajes (\textit{scores})}
$$\mathbb{Y} = \mathbb{X} \A$$
donde $\A = [\av_1|\av_2|\cdots|\av_m]$ y $m$ es el número de componentes seleccionadas.
\end{block}

\begin{block}{\textit{Biplot} (Gabriel, 1981; Gower y Hand, 1986)}
\begin{itemize}
  \item Representación bidimensional gráfica de individuos y variables.
  \item La longitud del vector representa la varianza.
  \item El ángulo entre los vectores representa la correlación.
\end{itemize}
\end{block}

\end{frame}
